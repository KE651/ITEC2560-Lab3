About one of my favorite programs...

For my Python programming course I wrote a program to create a mini travel guide for 5 Minnesota State Parks.
This was fun to work on because every time I ran it, I learned about different state parks!

Several functions divided the main operations of the program. The code shown below are excerpts only.
